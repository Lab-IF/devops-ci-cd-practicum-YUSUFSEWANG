# Refleksi Praktikum DevOps

Pada praktikum ini saya belajar mengenai instalasi dan konfigurasi tools yang digunakan dalam DevOps seperti Git, Docker, dan Visual Studio Code. Saya memahami bahwa DevOps bukan hanya sekadar tools, tetapi juga budaya kerja yang menekankan kolaborasi dan otomatisasi.

Harapan saya dari praktikum DevOps ini adalah dapat memahami proses CI/CD secara menyeluruh dan mampu melakukan deployment aplikasi secara otomatis menggunakan Docker dan pipeline.
Di akhir semester, saya ingin menguasai:
- Version control menggunakan Git secara profesional
- Containerization menggunakan Docker
- Membuat CI/CD pipeline sederhana
- Deployment aplikasi ke server atau cloud

Saya berharap ilmu ini dapat membantu saya menjadi software engineer yang siap kerja di industri.
