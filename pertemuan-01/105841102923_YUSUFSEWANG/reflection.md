# Reflection Praktikum DevOps

Pada praktikum ini saya belajar tentang pentingnya menyiapkan development environment sebelum memulai proyek.
Saya memahami bahwa DevOps bukan hanya tentang coding, tetapi juga tentang kolaborasi, automation, dan deployment.
Harapan saya dari praktikum ini adalah:
- Dapat memahami alur CI/CD
- Dapat menggunakan Docker dengan baik
- Dapat melakukan deployment aplikasi sendiri
- Menjadi lebih siap menghadapi dunia industri IT
Saya berharap dapat menguasai konsep DevOps secara praktis, bukan hanya teori.
